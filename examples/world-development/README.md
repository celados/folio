# World development acceptance component

Host-independent Folio acceptance component and build-time Resource shared by the TanStack Start and Astro examples.

The historical country data is derived from the official Observable
[Wealth & Health of Nations](https://observablehq.com/@mbostock/the-wealth-health-of-nations)
attachment and retained at SHA-256
`d98c992ae094d397f6bbd3b941cde5f32980f47d6e3707512ea951fbfe16b8a7`.

Based on free material from [Gapminder](https://www.gapminder.org/), used under
[CC BY 4.0](https://creativecommons.org/licenses/by/4.0/). The acceptance component changes the
source representation and query normalization, so it uses Gapminder's required modified-material
attribution.
